Wordpress 修改站点域名

发表于 2013-12-20   |  

最近在做一个Wordpress小站，但是发现用域名不能访问，用IP访问不能正确加载CSS、JS和图片，后果可想而知，最重要的是不能登录Wordpress后台了。

但是查了下DNS设置是没问题的，可以Ping到正确的IP。后来一番Google才想起来我们伟大祖国母亲再打我一次的备案制度——这个域名还没有备案。Wordpress的资源文件路径中的域名是不能访问的，所以不能加载。而且万网上说一般的备案流程需要20个工作日（干！）。

为了在这20天内可以用IP访问，可以通过设置Wordpress站点域名的方法来绕行。同样适用于更换域名。

所需要做的是：登陆phpMyAdmin，将wp-options（默认前缀是wp）中的siteurl字段修改为站点的公网IP即可。

Head First Bootstrap

发表于 2013-12-15   |  

如果你像痛恨GFW一样痛恨丑陋的页面，但自己却是CSS渣或设计无能，那么Bootstrap就是你的救星。

Bootstrap是Twitter开发的一套前端开发库，使用它可以快速开发Web2.0风格的页面，并且符合响应式设计。要使用Bootstrap，拢共分三步。

引入Bootstrap

可以在Bootstrap官网下载源码添加到自己的项目中，也可以在HTML中添加Bootcss提供的CDN。

使用CDN的好处是，服务器会自动选取最近的节点来提供文件的下载；另一方面，当用户之前浏览其他网页的时候如果加载过了这个文件，在访问我们的站点时就会从缓存中读取。

改造自己的HTML

只要为自己的HTML元素赋上特定的类名，就可以修改各个元素的外观甚至行为。

另外，Bootstrap还提供了响应式的导航条，会根据浏览器的分辨率自动调整样式，在移动客户端上会变成弹出式下拉菜单，效果很棒。使用Bootstrap的栅格系统，可以轻松完成页面的布局，并且可以自适应分辨率。

所有这些，都不需要写一行CSS或JS代码。

定制Bootstrap

当你用完爽过了之后，会发现所有用Bootstrap默认样式做出来的东西都是一副面孔，逼格瞬间下降了。这个时候就需要定制Bootstrap了。当然，已经有很多为Bootstrap开发的主题，比如Bootswatch。除此之外，还可以通过在这里自定义LESS变量来定制Bootstrap。设置完毕后下载编译好的CSS和JS文件即可。

Bootstrap自身提供了200个字体图标供我们使用。这些图标并不是图片，而是字体，可以不失真调整大小，自定义颜色、阴影等一切CSS能控制的样式。同样，只需要为<span>赋上特定的类名即可。如果丧心病狂的你认为这200个还不够用的话，我们还有Font Awesome，总有一款适合你。

HHKB上手

发表于 2013-11-29   |  

从土豪同学那借来了他刚买的HHKB。

我要找些字来打。

外形就不说了，的确非常好看。拿到手上的这块是黑色的，印字也是黑色的，看起来很低调的样子。

只是我在敲这些字的时候，会想去找方向键。Farbox编辑器不支持Ctrl+F、B、P、N的快捷键。

十几分钟打下来，主要感觉是：

优点：

  • 有人形容手感像“咬断一块巧克力”，说得还是比较贴切的。手感有点像薄膜键盘，只是更软，反馈更强一点，说像少女的肌肤什么的还是算了吧。
  • 按下去一点点就可以生效，按到底键帽撞击到底部的声音听起来挺闷的，像在敲一块木头。
  • Ctrl所在的位置非常舒服，不过这个通过软件也能做到。

缺点：

  • Mac下的调节屏幕亮度、多媒体控制等按键彻底不能用了。这也敢声称支持Mac？还是我没有找到设置的地方。
  • 当你想要找方向键的时候，你会要抓狂的（习惯之后说不定能适应）。Fn被安排在右下角的一个不起眼的位置，而方向键被放在几个符号的键位上，估计是为了可以单手操作。
  • 键盘底部的脚垫显得比较松垮，也没有类似Filco一样的橡胶防滑层，显得不够精致。
  • 有人说这块键盘可以刚好放在MBP的键盘上，我试了一下，尺寸是刚好，但是中出的那根线会直直顶上屏幕。放下来的话，触控板又不能用了，Mac的魅力少了一半。

1600软妹币呢，还是狠不下心来。再用个一两天吧。

Hello Farbox

发表于 2013-11-28   |  

很遗憾，我又在折腾了。

从一个同北邮的同名的哥们的博客上知道了Farbox。看上去就是把Octopress打包做成一个服务，这样就不用在各台机器上都配置一遍那个恶心的环境了。

更棒的是，可以直接用以前的markdown文件生成新的博客。

优点

  • 支持Markdown。
  • 可以自定义域名。
  • 可以自定义模板。
  • md文件都放在Google Drive或者Dropbox上，够稳定。
  • 有客户端可以用来编辑和同步。
  • 比直接使用Octopress要方便很多很多，不需要每次修改都生成一次，图片可以直接拖拽进来！！！

    缺点

  • 界面略丑，不过可以忍受。
  • Web端操作起来比较麻烦，有时候会报些用户不友好的错误。客户端用起来很舒服。

如果服务够稳定的话，我是愿意付费的。

希望Farbox能够茁壮成长。

Letter Press

发表于 2013-06-20   |  

最近在App Store发现了一款游戏，叫Letter Press。这款游戏获得了Apple的设计大奖，界面十分清新，动画生动自然，我非常喜欢。

这里是游戏截图：

游戏规则比较简单：

  • 两个玩家对战，轮流进行。玩家使用界面上的任意字母来组成一个个单词，使用过的字母将会变成特定的颜色（自己用过的变成蓝色，对方使用过的则会变成红色）。
  • 每次提交的单词不能是之前双方使用过的单词的前缀。比如之前提交过players，则不能再提交player或者play。
  • 如果某个字母以及它周围的字母全是自己方的颜色，这个字母就处于防御状态，对方下次使用这个字母时则不会变成对方的颜色。
  • 当所有的字母都被使用过，或某一个玩家拼不出单词的时候，获得字母多的一方获胜。

玩家可以在Game Center上找好友玩，也可以自动匹配玩家。

并且，这个游戏是免费的。

这里有一个玩这个游戏的小工具，感觉做得不错，有空的话再研究一下是怎么实现的。

Google Unchained

发表于 2013-05-28   |  

由于周所周知的原因，Google在中国常常处于不可用状态，经常会发现“连接被重置”的情况，即使侥幸打开了，点击搜索结果之后也需要很久才能打开，这是因为Google需要进行一些统计，结果中的链接不是真正的网址，需要连接Google服务器，之后再跳转到目标页面。

解决方法之一是翻墙，对于不爱折腾的用户，当然要找其他方法，总不能用百度吧。

今天在刷Google Reader时发现一款Chrome插件，叫急速谷歌，下面是功能介绍：

本插件会自动侦测可用的google服务器，在某服务器被屏蔽的瞬间切换到其他服务器。并将搜索结果中经由谷歌的链接转换为直接链接，从而提高访问速度和稳定度。

这里是插件的Web Store地址。

如果下载失败，可以使用这个技巧：

打开插件页面（如果连这个都打不开，你还是翻墙吧），看到地址栏里面的最后一段字符串，复制下来，替换到下面地址中“字符串”的位置，再把这个URL复制到其他浏览器中打开即可。

https://clients2.google.com/service/update2/crx?response=redirect&x=id%3D字符串%26uc

以急速谷歌为例：

插件地址：
https://chrome.google.com/webstore/detail/%E6%9E%81%E9%80%9F%E8%B0%B7%E6%AD%8C/nodbcjhlapenofbimeinhpojlcfaeobe
替换后的地址：
https://clients2.google.com/service/update2/crx?response=redirect&x=id%3Dnodbcjhlapenofbimeinhpojlcfaeobe%26uc

下载完毕后，打开Chrome的扩展管理页面，把文件拖进去就行了。
收工。
