Nested pop/push 导致的问题

发表于 2014-09-03   |  

有这样一个场景：当一个 ViewControllerA 出现的时候，我们检查某个状态值，如果符合条件就用 pushViewController 或 popViewController 跳转到另一页面。

很容易想到的方法就是在 ViewControllerA 的 viewDidLoad 或者 viewWillAppear 中来检查状态并跳转。但这时候系统会给出警告：

nested push animation can result in corrupted navigation bar.

Finishing up a navigation transition in an unexpected state. Navigation Bar subview tree might get corrupted.

或者

Unbalanced calls to begin/end appearance transitions for

并且可能导致 navigationController 的标题栏出现重影甚至崩溃。

导致这个问题的原因是：当一个 ViewController 的出现有动画时，viewWillAppear 和 viewDidAppear 是应该成对出现的，如果在 viewDidAppear 被调用之前 push/popViewController 会导致转换过程在 viewDidAppear 还没被调用时就终止，系统处在一个错误的中间态。

这里说的动画包括这个 ViewController 被 push 时有动画和下一个 ViewController 被 pop 时有动画。

解决方法显然有两种，一种是去掉动画；另一种是在viewDidAppear中调用push/popViewController，但是这样会导致动画完成时前一个画面会闪现一下才切换到下一个画面，目前还没有找到避免的方法。

ARC 简介

从 iOS 5开始，OC 引进了一种 ARC（Automatic Reference Counting）的方式来管理内存，由编译器来为我们生成相应的内存管理代码。

在类属性的声明中，ARC 相关的修饰词有两个：

  • strong 一个对象只要存在一个 strong 类型的指针指向它，就不会被释放。
  • weak 不会延长对象的生命周期，当weak类型的指针指向的对象被释放时，指针自身会被自动置为nil。

在 ARC 机制下，指针缺省的类型是strong。

在声明局部变量时，有四个 ARC 相关的修饰词，他们的用法跟 const 类似。

  • __strong 缺省值。用法同strong。
  • __weak 用法同weak。
  • __unsafe_unretained 类似于weak，但其指向的对象被释放时，其值不会被置为nil。应该是由于兼容性的原因而存在，通常不建议使用。
  • __autoreleasing 用来修饰一个以引用（id *）方式传入的参数，当函数返回值时被释放。

学车记

发表于 2014-04-29   |  

我报的是北京远航驾校，C1，周末速成班，富康，价格4400。从科目一第一次上课到拿到驾照，历时两个月整。

报名

报名的方式有很多种：学校代理、直接去报名点、找驾校教练。这里面的水有点深，我是找的学校代理，价格是4400，结果发现有一起学车的4200就能报上周末速成，还是比亚迪，说是赶上活动了，我顿时喷出一口老血。这个只能多打听打听了。

报名地点是五道口华清嘉园的门口。如果你色觉正常，通常报名点的人会帮你代办体检。而我，就得自己去一趟二炮了。所幸体检过了，乃们以后坐我的车系好安全带就是了。

至于富康还是比亚迪，我觉得富康也挺好。区别在于：富康的方向盘比较重，车屁股小，便宜100块。如果是力气小的女生，还是选比亚迪吧。
如果不差钱，还是选高冷的东方时尚吧，正规，服务质量也好。

科目一

远航的学员需要上三个半天的课，东方时尚的自己在家背直接去考就成，这里就体现了多掏的钱的价值了。第一次课要录考试用的指纹，后两次课都要点名，还是全点，这意味着每次都得去。

作为一个学生，最不需要担心的就是这种死记硬背的考试了。在手机上下一个名为“驾考宝典”的APP（跟我念：[æp]），先全刷一遍，再背背错题就没问题了。100道单选题，每题一分。90分就能及格，一般考下来都是98分左右。

科目二

约车

科目一考完自己回大厅约科目二的时间，一共6个半天，有两种选择：上午半天和下午半天。上午学车的话，得起个大早（北邮的班车是早上6点25，南门等车），但是还剩半天可以干点别的；如果是下午练车，一天就没了。作为一个床癌晚期患者，我果断选了下午。

练车

头一天一般是熟悉车内环境两个课时，剩下两个课时可以启动绕两圈。
这里有个小插曲，第一节课的四个课时当中，我都没有成功地放下过一次手刹。教练告诉我：你像这样握住手刹，拇指按下上面的按钮，往下掰就行了。我试了试，表示按不动按钮，不由得暗暗佩服教练的指力，两只手一起再试一次，还是按不动，不由得开始怀疑人生了。教练又重复了一遍。我还是不得要领，最终放弃回去上网查——先往上拉一点就行了，你说一声我不就知道了。

第二天起就是压离合控制车速加无尽的倒库了。教练会帮你调好后视镜，告诉你参照点，照着做就是了。

值得注意的是，镜子的位置很重要，你可以用手机模拟自己的视角拍下来，以后就可以自己调。我有一个半天就是因为镜子没调好倒库全跪，而上一个半天的倒库一次问题都没有出过。这对心态影响很大，还好及时发现，避免悲剧的再次发生。另外一点是，第一个参照点的打轮宁可早一点也不能晚，晚了就没有调整的余地了。

后面可能有一两次课是用来教另外4项的：定点停车和坡起、侧方停车、直角弯、S型弯道，在我眼里难度依次递减。这里就可以明白倒库的重要性了——科目二有一半以上的时间都是用来倒库的。坡起还是有点难度的，尤其是富康这种动力比较弱的车，需要油离配合，这个谁说了都没用，只能自己体会。

每次学车都是四个小时，实际练车的时间大概三个小时。速成班的时间都是安排好的，一旦约上了，少去一个半天就得交320的补课费。
即使是下午的课，一天下来还是挺累的，尤其是来回一共三个小时的班车坐下来，连我这种爱坐车的人也受不了。

考试

考试场地在海淀驾校旁边，就在京海教学楼斜对面。

驾校的车会送你到门口，然后听点名凭身份证进去。首先是在一个大厅里面等。大屏幕上会提示几号桩的前两个人是谁，等你名字后面的排序变成1就说明轮到你了。

考试规则是，每人有两次机会，当你其中一项挂了之后，之前考过的每一项都要重考一遍。第一项就是倒库，因为最难所以放在最前面，如果挂了的话，只需要重考倒库这一项就行了。

当时在考场的铁栅栏外等待的时候，还是挺紧张的，尤其是你看到一个哥们连跪两次的时候。考完倒库，再回到大厅等着，接下来的四项是一次性完成的。顺序是：定点停车和坡起、S弯道、侧方、直角弯。点到名字之后去找对应号码的车和协考。我做了个考场的地图。需要注意的是，S弯在一个棚子里面，所以如果外面阳光很强，眼睛就有个明暗适应的过程。

听话就能过。考试过程太快，一点感觉都还没有就结束了。

不过据说4月以后科目二考试车上不准坐协考了，应该会有别的方式帮过吧，驾校总是要保证大多数人都能考过的。

考完回大厅签字，约科三的课。

科目三

练车

一共三次课，两次白天，一次夜间训练。

主要学的内容有：加减挡、并线、转弯、掉头、靠边停车。整个科目二期间，我只挂过一次二挡，科目三终于上五挡了。后来发现，练车的地点就是考场，教练会带你把三个考场（北一、北二、聂东）都练一遍。

必须逐级加挡，可以越级减挡。加挡时，我是先踩油门，等转速到了1500到2000时就加一挡。新手在高挡的时候就不知道如何减速，一踩刹车就不自觉地踩离合，这是科目二留下的后遗症——怕熄火。其实如果车速不是太慢，根本没有那么容易熄火，放心大胆地踩刹车吧，等车速降下来再踩离合，根据车速降到合适的挡位。

夜间训练就很扯了，先给你讲讲灯的开关都在哪，然后给一张纸让你看各种口令下怎么打灯，最后跟白天一样绕几圈就结束了。我练车的时候天还是亮的，什么灯都没开。

考试

考试也是两次机会。夜考是随机抽取20%的人参加，去考试的那天才知道有没有被抽中。反正我是没有被抽中，据说白天考试过了，夜考就没有挂的。

白天考试流程是：先去驾驶位一侧的车窗外喊”报告“，逆时针绕车一周再喊报告，上车双手递上身份证，然后调座椅、系安全带，轰一脚油门再说”仪表检查完毕，请求起步“。考官一共只会发四个口令：起步、掉头、掉头、靠边停车。整个过程就是在一条笔直的马路上绕一圈回来，想想就不难吧。

秘诀是：

  • 记得打灯。
  • 一定要看后视镜，就像你打Dota看小地图一样。起步要看，并线要看，没事就看；看的时候要有明显的扭头动作。
  • 换挡不能看挡把，看了就跪。
  • 二挡转弯，一挡掉头，虽然考试过程中没有转弯。
  • ”车要走起来“。尽可能快地上五挡，速度起码50以上。如果前面的车太慢，可以超车。等到考官提醒你”快点“的时候就危险了。

科目四

这个时候驾照基本上就已经到手了。科目四的考试题型是50道选择题，其中5道多选，每题2分，90分及格。考试内容跟科目一差不多，考前背了一个晚上就满分过了。

领证

远航的官方说法是考完之后的第一个周末早上去领驾照，就意味着还得起个大早花半天时间。但是，班车队队长告诉我们说考试当天下午4点多就可以在约车大厅领了。我们一行三个人趁机去凤凰岭玩了一圈。

下午驾照到手。

题外话

关于买烟。当你在外面练车中途休息的时候，会有各种大妈过来问你要不要买烟，你可以直接回绝，不用担心，以我的经验来看，教练的脸也没有黑到哪里去。有一同去的学妹练科目二撞石头上了，也是自己买烟带过去的。

关于教练态度。人和人之间的差别真的是太大了，有的张口就骂，有的不闻不问，也有慈祥的大叔。但是大多数情况是嘴上不饶人，不知道是因为北京人与生俱来的优越感，还是因为教练的生活太艰辛。
具体表现是：每一句话都是反诘。

我能上车吗？
上来干嘛啊？

这后视镜没用呗？没用我把它掰了得了。

回头想想，学车的过程虽然辛苦，但还是挺有意思的，大家一起讨论驾车技巧，分享考试经验，也见识了各种嘴脸。

现在只盼土豪能借车给我过过瘾。

北邮办理护照攻略

发表于 2014-01-09   |  

最近试图考驾照，但是报名时被告知需要有北京地址的身份证。如果已经有北京户口，就要办临时身份证；如果是外地户口，就要办狗日的暂住证。

办临时身份证或者护照，需要在学校保卫科借户籍卡。于是悲剧开始了。

大致是这个样子的：

  • 写申请，找导师签字。（如果是办护照，就需要在北邮人主页提交申请。）
  • 找辅导员签字。
  • 到学院办公室盖章。
  • 到小白楼204的学生处盖章。
  • 带上学生证、身份证和申请到保卫处借户籍卡。慢着，他们只在上午8:30到11:30可以办理户籍卡借出！
  • 到北太平庄派出所办理。（如果是护照，这里有一个攻略。万能的北邮人！）

如果你恰好需要借户籍卡，那么最好顺便把护照办了，因为据说在学校办理还是相对容易的。万一哪天打通任督二脉，能够肉身翻墙，可以用得上。

今天跑了一圈，只拿到了导师签字。辅导员说下周一才来，非得亲眼看到导师签字，不能代签。

教训是，签字盖章这种事，你永远不知道有多少依赖项在等着你，那帮大爷们是个什么样的工作时间。我能做的就是：事先调查清楚；尽量早地开始准备；做好折腾的打算。

今天领到了临时身份证、办好了港澳通行证和护照。

有几个注意事项：

  • 可以同时办理港澳通行证和护照，因为步骤是一样的，而且都在一幢名为“中关村.+服务中心”的大楼中办理。
  • 办理上面两样需要身份证和户籍卡上的地址一致。如果不一致，你就需要办理临时身份证。
  • 仅仅需要户籍卡、身份证、两寸照片两张（也可以去那边临时洗，30块）。

所以办理的过程是：

首先，到北太平庄派出所办理临时身份证。从学院路一致往南走过了今典花园不远就到了。去了先照相，希望你不会也后悔为什么当时头发搞得那么挫。工作人员会给你一张纸条，让你去“中关村.+服务中心”去取临时身份证。正式身份证十个工作日后在北太平庄派出所领。

到明光桥南坐392到底站“西钓鱼台”，一下车就是“中关村.+服务中心”，不用担心找不到。去三楼取个号排队。在116号窗口领临时身份证，大约要等10分钟左右，期间可以填写护照的申请表。

取到临时身份证后，拿着填好的表格去拍照，因为需要电子档录入系统。这里的拍照是不能戴眼镜或镜框的。接下来再去取号、等待、办理。

最后交钱，港澳通行证160，护照200。过一段时间后再来取，也可以选择邮寄。

所有的手续都在三楼办好，不得不说这里的办事效率还是挺高的，办理这些我一共花了45分钟。

记一个小网站的诞生

发表于 2013-12-28   |  

最近经萌萌介绍，接了一个做网站的活。那人要求很低，说是只需要静态页面，但是后期能自己添加东西就行。由于之前折腾过Joomla，就斗胆接下来了。

很快，萌萌把网站内容等材料发过来了。我一看，这尼玛就是坑爹啊！网站Logo丑就不说了，内容还是全抄别人的，连人家的公司名都没有改全！图片质量很渣，还有水印！顿时信心大增。

由于静态页面制作起来比较麻烦，而且后期还不好编辑、添加内容，这次决定用Wordpress，跟Joomla差距应该不大。

第一夜

主机用的是阿里云而不是之前用的虚拟主机，还需要自己搭环境。拿到账号密码，找了一个教程，从硬盘格式化开始，按部就班地来。

安装配置Apache、MySql、vsftp、Wordpress、phpMyAdmin，尤其是vsftp，弄得我焦头烂额，Linux渣伤不起。后来回宿舍，哥们提醒，直接用scp就OK了，压根不用这玩意，我顿时一口老血喷出。再后来发现，ftp还是需要配置的，因为在wordpress后台安装主题或插件时需要用到。

第二夜

找模板。

模板要配色清爽，简单大气，还要高度可自定义。找了好久才找到Customizr。看名字就知道好改。简单了解了下Wordpress的“仪表盘”（就是后台页面），把网站的内容都填了进去。看到网站就这么搭起来了，心中暗爽。

回来给Show给小伙伴看的时候掉链子了，发现域名不能用，于是有了这么一出。

第三夜

找图片。

首页的大Slider需要三张大图，于是在Google之。此时脑中立马浮现出一群衣冠楚楚的老外（一定要有个黑人在里面），傻呵呵地握手看着镜头傻笑的场面，就是这货没跑了，总觉得这些老板就好这口。另外还找了几张小图片，是那种抽象立体小人。当然这里的图片用的都是盗版的，鄙视我吧。

另外还用我仅有的一点点CSS知识改了下排版。

心想，这下一定完爆你的要求了。

第四夜

我让萌萌把网站给客户看了。萌萌说客户还比较满意，但是还有几个地方要改。

发来了一个Word。打开一看，顿时惊了。里面齐刷刷躺着8条修改意见，其中几条说得是页面上的样式设计问题，说得我真没脾气。痛定思痛，按照意见一条条修改下来，确实好看了许多。

第五夜

客户发现了两处Bug，一个是导航栏尺寸不对，另一个是合作伙伴的图片尺寸大小不对。看了客户发来的截图，原来他用的是360浏览器，貌似是IE8内核。在萌萌的帮助下，顺利解决。
